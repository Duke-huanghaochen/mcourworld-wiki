# Ourworld 维基

这里是 **Ourworld 私人服务器** 的官方维基源码仓库。

## 📖 关于
Ourworld 是一个邀请制的 Minecraft 私人生存服务器，采用极限生存规则（第一次死亡可复活，第二次死亡封禁一年）。本维基记录了服务器的宪章、教程、历史故事和更新日志。
